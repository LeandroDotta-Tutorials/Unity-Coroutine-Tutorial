# Exemplos

O projeto possui 6 exemplos diferentes de diferentes usos de corrotinas

## Exemplo 1 - Ação Única Com Delay

- **Cena**: 01-Coroutine_SingleActionWithDelay
- **Descrição**: Cria uma corrotina para executar uma ação após alguns segundos.

**_OBS_**_: Este exemplo possui 3 implementações (Sem Corrotina, Com Método Invoke, Com Corrotina). Cada implementação está separada em uma seção `#region` diferente. Basta comentar uma implementação e descomentar outra, para usá-la._

## Exemplo 2 - Ações em Intervalos

- **Cena**: 02-Coroutine_ActionInIntervals
- **Descrição**: Cria uma corrotina para executar ações em intervalos de segundos.

**_OBS_**_: Este exemplo possui 3 implementações (Sem Corrotina, Com Método Invoke, Com Corrotina). Cada implementação está separada em uma seção `#region` diferente. Basta comentar uma implementação e descomentar outra, para usá-la._

## Exemplo 3 - Cadeia de Ações

- **Cena**: 03 - Coroutine_ActionChain
- **Descrição**: Cria uma rotina que executa uma cadeia de ações

## Exemplo 4 - Cooldown

- **Cena**: 04 - Coroutine_CooldownSample
- **Descrição**: Cria uma corrotina simulando um efeito de Cooldown

## Exemplo 5 - Multiplas Corrotinas

- **Cena**: 05 - Coroutine_MutipleCoroutines
- **Descrição**: Cria multiplas corrotinas para demonstrar que é possível ter várias corrotinas rodando ao mesmo tempo em um GameObject.

## Exemplo 6 - Requisição HTTP

- **Cena**: 06 - Coroutine_HTTPRequest
- **Descrição**: Cria uma corrotina que faz uma requisição HTTP em paralelo.
